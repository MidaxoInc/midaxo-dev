ASP Forecast

The ASP Forecast uses the Average ASP per quarter from the trailing 4 quarters (blue line). The green bars indicate actual ASP per quarter, or average open deal value for future quarters.

Existing Pipeline Forecast

The Existing Pipeline forecast is calculated as open deals x real win rate for each stage.

  • Open Deals: Deals with a close date in quarter, excluding deals created in quarter.
  • Real Win Rate: This is calculated as deals won in quarter / open deals (by stage). In other words, it excludes deals that are closed lost and deals that were pushed into another quarter.

Create & Close Plug

Deals expected to be created & closed in quarter is calculated as 1 / (1- % won deals created in quarter) * existing pipeline forecast.

  • % of won deals created in quarter is calculated using a the average rate from the previous 2 quarters.
